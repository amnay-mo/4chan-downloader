# 4chan downloader
Python script for downloading images from 4chan threads

## Requirements
- python 3.5+
- tqdm
- aiohttp
- urllib3

## Usage
```
usage: 4chan_dl [-h] --url URL [--dir DIR]

downloads all images from a 4chan thread

optional arguments:
  -h, --help  show this help message and exit
  --url URL   Thread URL
  --dir DIR   Output Directory
```
