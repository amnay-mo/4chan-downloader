# 4chan-downloader
Python script for downloading images from 4chan threads

## Requirements
- Python 2.7.x

## Usage
`python 4chan_dl.py -u <thead_url> -o <output_directory>`
## Example
`python 4chan_dl.py -u http://boards.4chan.org/m/thread/14569227/gm-thread -o ./4chan/gm`
